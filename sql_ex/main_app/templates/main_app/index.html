Hello and welcome to the sql_ex front end!

This page serves as the homepage/index of the sql_ex app: main_app

The URL endpoint is defined in the urls.py file. In this case it is '', indicating the root of main_app/. Endpoints always point to views which serve the content/functionality of the endpoint, defined in views.py. This page is a regular view, meaning it focuses on the front end by rendering HTML content for user interaction. It provides a graphic user interface (GUI) where users can navigate and access different features of the app.

In contrast, API views handle back-end operations, such as data retrieval and manipulation. For instance, when a user clicks the button below to fetch a list of records, the regular view (which renders this HTML file - index.html) calls the API endpoint /main_app/api/recordlabel/ to fetch the necessary data using main_app.js (JavaScript). The API endpoint then points to the associated API view RecordLabelListApiView to process the request, interact with the database model RecordLabel, and return the serialized response in JSON format. The JavaScript function that called the API endpoint then processes the response as needed, and displays it on the HTML page.
